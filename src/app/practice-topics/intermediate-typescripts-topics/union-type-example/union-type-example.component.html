Union Types in TypeScript

Union types allow a variable to hold more than one type.

What is a Union Type?

A union type allows a variable to accept multiple types using the pipe (|) operator.

This is useful when a value can logically be more than one type. 

{{ userTypeCode }}
Why use Union Types?
  • To handle flexible or dynamic data (API responses).
  • To avoid using any while keeping type safety.
  • Commonly used with form inputs and backend values.

When working with union types, TypeScript allows type narrowing using conditions like typeof.

Key Takeaways
  • Union types use the | operator
