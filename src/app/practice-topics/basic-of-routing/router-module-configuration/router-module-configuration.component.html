Basics of Routing in Angular

Learn how Angular navigation works using RouterModule and Routes.

What is Angular Routing?

Angular routing allows you to navigate between different views (components) without reloading the page. It is handled using RouterModule and a Routes configuration.

RouterModule Configuration

RouterModule is responsible for setting up navigation. In traditional Angular apps, it is configured in a routing module.

pathMatch tells Angular how to match the URL.

  • full → full URL must match
  • prefix → URL starts with the path (default)
