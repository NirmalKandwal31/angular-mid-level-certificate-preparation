RouterOutlet & routerLink

Display routed components and navigate between routes.

What is Angular Routing View?

Angular uses <router-outlet> as a placeholder where components are loaded based on the active route. Navigation between routes is done using routerLink.

<router-outlet>

<router-outlet> acts as a container where Angular injects the component that matches the current route. 

{{ routerOutletCode }}
Only one routed component is displayed inside a router-outlet at a time.
[routerLink]

routerLink is used to navigate between routes without reloading the page. 

{{ routerLinkCode }}
  • Works like href but for SPA navigation
  • Prevents full page reload
  • Keeps application state intact
routerLinkActive

routerLinkActive automatically applies a CSS class when the linked route is active. 

{{ routerLinkActiveCode }}

This is commonly used to highlight the active menu item.
