What is FormControl?

FormControl represents a single input field in Reactive Forms. It tracks the field value, validation state, and user interaction state.

setValue vs patchValue

setValue is strict (mainly matters for FormGroup), while patchValue is flexible for partial updates. For a single FormControl, both will update the value.

Method Behavior
setValue() Strict for complex forms (FormGroup). Requires full structure match.
patchValue() Allows partial updates (very useful in FormGroup).
valueChanges

valueChanges is an Observable that emits whenever the control value changes. Useful for live search, auto-save, dynamic validation messages, etc.
