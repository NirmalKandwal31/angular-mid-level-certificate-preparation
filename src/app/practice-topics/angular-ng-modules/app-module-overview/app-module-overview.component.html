Root Module (Older Angular Structure)

In the traditional Angular architecture, AppModule is the root NgModule. Angular loads this module first and uses it to configure the entire application.

What does "bootstrap: [AppComponent]" mean?

The bootstrap property tells Angular which component should be loaded first when the application starts.

Application Startup Flow
  1. Browser loads index.html.
  2. Angular loads main.ts.
  3. AppModule is loaded as the root module.
  4. AppComponent is bootstrapped.
  5. <app-root> is rendered inside index.html.
