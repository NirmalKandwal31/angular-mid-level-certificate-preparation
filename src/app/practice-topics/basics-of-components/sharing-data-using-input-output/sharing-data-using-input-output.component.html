Data Flow (Easy)
1) Parent → Child
  • Parent passes a value using Input().
  • Child receives it in its TS file.
  • Child shows it in HTML.
2) Child → Parent
  • Child emits an event using Output().
  • Parent listens in HTML using (event)="handler($event)".
  • Parent updates its own variable and UI updates.
