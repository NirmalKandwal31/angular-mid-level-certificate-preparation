Spread Operator (...)

The spread operator expands arrays/objects into individual values and helps copy/merge data in a clean way.

What is the Spread Operator?

The spread operator (...) expands an array or object into individual elements/properties.

Why do we use it?
  • To merge arrays quickly (without loops).
  • To clone arrays/objects in an immutable way.
  • To create updated copies of objects (common in Angular state updates).

Key Takeaways
  • ... (spread) expands values to merge/copy.
  • Great for immutable updates (create a new array/object instead of mutating).
  • Spread is a shallow copy, so for nested objects spread nested levels too.
