Attribute Directives

Learn how ngStyle and ngClass dynamically change the appearance of elements.

What are Attribute Directives?

Attribute directives change the appearance or behavior of an existing DOM element without changing the DOM structure.

Key Differences
  • ngStyle → modifies inline styles
  • ngClass → adds/removes CSS classes
  • Both are attribute directives
  • They do not add or remove DOM elements
