HostListener & HostBinding

Interacting with the host element directly from a component.

What are HostListener and HostBinding?

HostListener allows a component to listen to DOM events occurring on its host element.

HostBinding allows binding a property of the host element (like style, class, or attribute) directly from the component.

Key Differences
  • HostListener → listens to events
  • HostBinding → binds properties
  • Both work directly on the host element
  • Commonly used in directives and reusable components
