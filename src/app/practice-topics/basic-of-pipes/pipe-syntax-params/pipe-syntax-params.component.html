Chaining

You can chain pipes by writing multiple pipes. Output of one becomes input of the next.

Original: {{ name }}
Chained: {{ name | titlecase | slice:0:8 }}

Key Takeaways
  • value | pipe is the base syntax.
  • value | pipe:arg1:arg2 passes parameters.
  • Chaining applies multiple transformations step-by-step.
