Common Angular Pipes

Pipes format and transform data directly in the template.

What are Pipes?
  • {{ d }}
Date Pipe

Formats date values into readable date strings.

Output: {{ today | date:'fullDate' }} 
{{ dateCode }}
Currency Pipe

Formats numbers as currency based on locale.

Output: {{ price | currency:'USD' }} 
{{ currencyCode }}
Percent Pipe

Converts a number into percentage format.

Output: {{ discount | percent }} 
{{ percentCode }}
JSON Pipe

Displays objects in JSON format (very useful for debugging). 

{{ user | json }}
{{ jsonCode }}
Slice Pipe

Extracts a portion of an array or string.

Output: {{ f }}, 
{{ sliceCode }}
Key Takeaways
  • date → formats dates
  • currency → formats money values
  • percent → formats percentages
  • json → displays objects for debugging
  • slice → extracts part of array/string
