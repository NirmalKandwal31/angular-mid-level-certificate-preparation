When to Use Pipes

Pipes are for formatting view data — avoid heavy logic in templates.

What are Pipes best for?

Pipes are ideal for display formatting — transforming values right before showing them in the UI (date, currency, percent, case changes).

Avoid Heavy Logic in Template

Templates run during change detection. Expensive work inside templates (heavy filters, sorts, complex calculations) can slow down your UI.
