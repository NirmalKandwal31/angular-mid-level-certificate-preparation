What is a Custom Pipe?

A custom pipe is a reusable formatter that transforms values in templates. You define it using the Pipe decorator and implement PipeTransform with a transform() method.

Pipe Decorator

The Pipe decorator registers your class as a pipe. The name is what you use in templates.

Key Observation
  • Push (Mutate) keeps the same array reference → pure pipe may not update.
  • Replace Array creates a new reference → pure pipe updates.
  • Impure pipe updates even on mutation — but runs more often.
Key Takeaways
  • Pipe registers your pipe and gives it a template name.
  • transform() implements the transformation logic.
  • Use pure pipes by default (best performance).
  • Use impure pipes only when required.
